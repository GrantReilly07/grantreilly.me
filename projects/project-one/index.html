Raspberry Pi LED matrix

LED Matrix Editor

A virtual 32x32 LED matrix editor for designing frames before they are sent to a Raspberry Pi controlled LED display.

Draw a Frame

Each square maps to one LED pixel. The editor stores the frame as 32 rows of 32 hex color strings for the future backend API.
